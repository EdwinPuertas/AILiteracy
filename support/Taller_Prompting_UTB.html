Técnicas de Prompting con IA

Aprende cuándo y cómo usar Zero-Shot, One-Shot y Few-Shot Prompting para obtener resultados precisos y consistentes en tu trabajo universitario.

Zero-Shot

Haces la tarea directamente sin dar ningún ejemplo. La IA usa solo su entrenamiento para inferir el formato esperado.

Ideal para tareas simples, preguntas directas o primera exploración.

0 ejemplos

One-Shot

Proporcionas exactamente 1 ejemplo de entrada y salida antes de tu tarea real. Establece el patrón mínimo.

Ideal cuando el formato es específico y la IA puede inferirlo con un solo modelo.

1 ejemplo

Few-Shot

Das 2 a 5 ejemplos que cubren variaciones del patrón. La IA generaliza el formato con mayor precisión.

Ideal para comunicaciones institucionales, análisis de datos o tareas con formato muy específico.

2–5 ejemplos

Anatomía de un prompt con shots

Todo prompt de calidad tiene cuatro componentes. Los shots son el componente que diferencia las técnicas:

1. Rol
Eres asistente de Registro Académico de la UTB
2. Contexto
Respondemos solicitudes estudiantiles con tono formal institucional
3. Shots
Entrada 1 → Salida 1 / Entrada 2 → Salida 2 (0, 1 o varios)
4. Tarea real
Ahora responde esta nueva solicitud: [caso concreto]

¿Cuándo usar cada técnica?

Zero-Shot · sin ejemplos

  • Preguntas directas y factuales
  • Resúmenes generales de texto
  • Brainstorming inicial de ideas
  • Tareas sin formato específico
  • Clasificaciones con criterios claros

One-Shot · 1 ejemplo

  • Respuestas con estructura fija
  • Clasificaciones simples (sí/no)
  • Adaptación de estilo de escritura
  • Formatos de datos sencillos
  • Reportes con plantilla única

Few-Shot · 2 a 5 ejemplos

  • Comunicaciones institucionales
  • Análisis de datos con formato
  • Clasificación con matices
  • Redacción con voz de marca
  • Fichas o reportes estandarizados
