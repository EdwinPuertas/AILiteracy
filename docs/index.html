Líneas de investigación

Una ruta de aprendizaje conectada con proyectos reales de investigación y aplicación industrial.

Natural Language Processing

Modelos transformer para clasificación de textos, análisis de sentimientos, detección de propaganda, discurso de odio y framing mediático.

Computer Vision

Sistemas basados en CNN para agricultura de precisión, detección de enfermedades en cultivos, monitoreo de tráfico y smart cities.

Orientación vocacional con IA

Sistemas de recomendación para apoyar a estudiantes de bachillerato en la elección de carrera y reducir la deserción universitaria.

Cómo usar este repositorio

Clona el proyecto, explora los materiales y abre los notebooks desde la carpeta de soporte.

Comandos rápidos

# Clonar el repositorio
git clone https://github.com/EdwinPuertas/AILiteracy.git
cd AILiteracy

# Explorar materiales de clase
ls general/

# Abrir notebooks
jupyter notebook support/

Requisitos mínimos

  • Python 3.8+
  • Jupyter Notebook / JupyterLab
  • transformers, scikit-learn, pandas, matplotlib

pip install transformers scikit-learn pandas matplotlib jupyter
